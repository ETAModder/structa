# ElementSizes
In Structa, most elements have three sizes: normal, small, and fine print.  
You put in the size after the $ in each line. so $1 = normal, 2 = small, and 3 = fine print.  

```
{$1"Normal header"
{$2"Small header"
{$3"Tiny header"

[$1"Normal text"
[$2"Small text"
[$2"Fine print"
```

This works in images and links too.
